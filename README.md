# 🗡️ Minecraft Item Command Generator 
> 一个基于 Python 的交互式物品指令生成器，轻松制作神级装备！

[![Python](https://img.shields.io/badge/Python-3.6+-blue?style=flat-square&logo=python)](https://www.python.org/)

## 📖 项目简介
这个工具旨在帮助玩家快速生成 Minecraft 复杂的 `/give` 指令。你不需要背诵冗长的 NBT 标签，只需按照程序提示选择物品、附魔和等级，脚本就会自动为你拼装好指令。

### 🌟 核心亮点
* **防错机制**：内置附魔互斥逻辑（例如：保护和火保护不能共存），确保生成的指令在游戏中有效。
* **高度自定义**：支持自定义物品显示名称、附魔等级和物品数量。
* **多模式运行**：既可以在本地终端（Terminal）交互运行，也可以通过 GitHub Actions 网页端生成。

## 🚀 本地使用指南

### 1. 准备环境
确保你的电脑安装了 **Python 3最新版**。你可以通过 `python --version` 查看。

### 2. 获取代码
git clone就行了，有手就行

### 3. 运行代码
双击打开或
python generator.py

### 4. 使用
按照程序提示输入即可
